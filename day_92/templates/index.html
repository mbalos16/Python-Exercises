HueHarvest

This image palette generator employs the k-means clustering algorithm to analyze an image's color data, identifying the 10 most prevalent colors. This process involves sampling the image's pixels, using the k-means algorithm to find clusters in the color space, and then displaying the resulting palette for visual appreciation and further use.
